# Getting Started

Clone repository and `cd` into infor_mn. Run `npm start` and the app will run in development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### **_Technologies Used_**

- React.js
- Material UI
- Axios
